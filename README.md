# Come calcolare il codice fiscale

Il **codice fiscale italiano** si calcola partendo dai dati anagrafici: nome, cognome, data di nascita, sesso e luogo di nascita. È un codice di 16 caratteri.

---

## 1. Cognome (3 lettere)

Si prendono le **prime tre consonanti** del cognome:

- Rossi → RSS  
- Bianchi → BNC  

Se le consonanti non bastano, si aggiungono le vocali, poi eventualmente “X”.

---

## 2. Nome (3 lettere)

Si prendono le consonanti:

- Se sono **4 o più**, si prendono **1ª, 3ª e 4ª**  
  - Marco → MRC  

- Se sono meno di 4, si segue la stessa regola del cognome.

---

## 3. Anno di nascita (2 cifre)

Le ultime due cifre dell’anno:

- 1990 → 90  
- 2005 → 05  

---

## 4. Mese (1 lettera)

| Mese      | Lettera |
|-----------|--------|
| Gennaio   | A |
| Febbraio  | B |
| Marzo     | C |
| Aprile    | D |
| Maggio    | E |
| Giugno    | H |
| Luglio    | L |
| Agosto    | M |
| Settembre | P |
| Ottobre   | R |
| Novembre  | S |
| Dicembre  | T |

---

## 5. Giorno + sesso (2 cifre)

- Uomini: giorno normale (es. 15 → 15)  
- Donne: giorno + 40 (es. 15 → 55)

---

## 6. Comune o stato di nascita (4 caratteri)

Ogni comune italiano ha un codice (es. Roma = H501).  
Per l’estero esistono codici specifici.

---

## 7. Carattere di controllo (1 lettera finale)

È una lettera calcolata con un algoritmo basato sui 15 caratteri precedenti.

---

## Esempio

Mario Rossi, nato il 15 luglio 1990 a Roma:

- Cognome: RSS  
- Nome: MRA  
- Anno: 90  
- Mese: L  
- Giorno: 15  
- Comune: H501  

→ **RSSMRA90L15H501 + lettera finale**
